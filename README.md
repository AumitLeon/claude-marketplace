# claude-marketplace

> **Note: This is a test repository.**

A Claude Code plugin marketplace by Aumit Leon.

## Plugins

### quality-review-plugin

Adds a `/quality-review` skill for quick code reviews.

Reviews your selected code or recent changes for:
- Potential bugs or edge cases
- Security concerns
- Performance issues
- Readability improvements

## Installation

Install via Claude Code:

```
/install-plugin https://github.com/aumitleon/claude-marketplace
```
